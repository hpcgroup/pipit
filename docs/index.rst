#####
Pipit
#####

Pipit is a Python library for analyzing parallel execution traces.

You can get pipit from its `GitHub repository
<https://github.com/hpcgroup/pipit>`_:

.. code-block:: console

  $ git clone https://github.com/hpcgroup/pipit.git
